To run application follow these steps:
1. Execute command 'mvn clean install' on $root folder.
2. Execute command 'mvn clean install tomcat7:run' on '$root/crental-web' folder, to run web application and rest services. Web app is then available on 'http://localhost:8080/pa165/'
3. Execute command 'mvn clean install tomcat7:run' on '$root/crental-client' folder, to run rest client. Rest client is then available on 'http://localhost:9966/crental-client/'
